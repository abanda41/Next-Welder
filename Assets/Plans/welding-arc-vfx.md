# Project Overview
- Game Title: VR Welding Simulator
- High-Level Concept: A realistic welding simulation for VR, focusing on visual fidelity and accurate physics.
- Players: Single player (VR)
- Inspiration: Professional welding training simulators.
- Tone / Art Direction: Realistic, Industrial.
- Target Platform: PC VR (Windows Standalone)
- Render Pipeline: URP (Universal Render Pipeline)

# Game Mechanics
## Core Gameplay Loop
The player uses a VR welding torch to join metal pieces. The core experience relies on the visual feedback of the arc, the weld bead formation, and the sound of the welding process.
## Controls and Input Methods
VR Controllers for torch position and orientation. Trigger for arc activation.

# UI
In-game HUD (floating in VR) showing Work Angle, Travel Speed, and Stickout Distance.

# Implementation Steps

## 1. Arc Core Jitter & Flicker
- **Objective**: Make the arc core look like unstable plasma.
- **Node Setup**:
    1. Open `ParticleEffect.vfx`.
    2. In the `ARC CORE` system, locate the `Set Color` or `Set Attribute: Color` block.
    3. Add a `Periodic Total Time` node.
    4. Connect it to a `Multiply` node (Frequency: ~30-60) then to a `Perlin Noise 1D` node.
    5. Use the noise output (remapped to [0.8, 1.2]) to multiply the `Intensity` of the HDR Color.
    6. For **Position Jitter**: Use a similar noise setup (3D noise) to add small offsets to the `Position` attribute in the `Update` block.
- **Dependency**: Existing `ARC CORE` system.

## 2. Atmospheric Cyan Glow (Bloom)
- **Objective**: Create the large cyan halo seen in the reference image.
- **Node Setup**:
    1. Add a new `Simple Swarm Particle System` or `Simple Quad System` block.
    2. **Spawner**: Constant rate of 1 (or a single persistent particle).
    3. **Initialize**: Set lifetime to infinite.
    4. **Output**: 
        - Use a `Radial Gradient` or `Smoothstep` texture.
        - Set `Size` to be large (~0.5 - 1.0 units).
        - Set `Color` to a Cyan/Light Blue HDR color (e.g., #00FFFF with Intensity 5-10).
        - Set `Blend Mode` to `Additive`.
    5. **Flicker Sync**: Link its intensity/alpha to the same noise node used for the core flicker but with a lower amplitude.

## 3. Spark/Spatter System
- **Objective**: Add physical sparks that fly off the arc point.
- **Node Setup**:
    1. Add a new `Constant Rate` spawner to the graph.
    2. **Initialize**: 
        - `Set Velocity`: Random in a Cone (oriented towards the surface normal).
        - `Set Lifetime`: Random (0.2 to 0.6s).
        - `Set Size`: Very small (0.01 to 0.03).
    3. **Update**:
        - `Gravity`: Add a small downward force.
        - `Linear Drag`: To slow them down over time.
        - `Collision (AABB or Plane)`: If a surface plane is defined, make them bounce or die on impact.
    4. **Output**:
        - Use a `Stretched Billboard` or `Particle Strip` for a motion-blur look.
        - Color: Bright Orange/Yellow HDR.

## 4. Heat Distortion (Air Shimmer)
- **Objective**: Simulate the heat generated by the arc.
- **Node Setup**:
    1. Add a small system spawning soft quads.
    2. **Shader**: Use a shader that samples the camera's Opaque Texture with an offset driven by a scrolling noise texture.
    3. Set the `Render Queue` to `Transparent` and ensure it's rendered after the world but before the UI.

## 5. Dynamic Light Integration
- **Objective**: Make the arc light up the environment.
- **Action**: 
    1. Add a `Point Light` component to the welding torch prefab at the arc's origin.
    2. Use a script (or VFX Graph property binding) to sync the light's intensity and color with the VFX's flicker noise.

# Verification & Testing
- **Visual Check**: Run the scene in the Editor. Ensure the core is "blindingly" bright (white) and the cyan halo is visible.
- **Flicker Check**: Verify the flicker speed feels natural (fast but not seizure-inducing).
- **VR Performance**: Test in the VR headset to ensure the high particle count doesn't drop the frame rate.
- **Bloom Check**: Ensure the Post-Processing Volume has `Bloom` enabled with a high threshold so only the arc glows.
